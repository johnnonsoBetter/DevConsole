Features

Everything you need to debug faster

DevConsole brings powerful AI capabilities and seamless integrations directly into your Chrome DevTools.

AI-Powered Log Analysis

Get instant, intelligent explanations for any console log. Understand errors, warnings, and debug messages with AI that speaks developer.

GitHub Issue Automation

Create perfectly formatted GitHub issues directly from your console errors. AI enhances your issues with context and reproduction steps.

GraphQL Explorer

Full-featured GraphiQL interface built right into DevTools. Explore schemas, run queries, and test mutations without leaving your workflow.

Smart Autofill

Fill forms instantly with intelligent field detection. Supports custom datasets and integrates with Unsplash for beautiful placeholder images.

Enhanced Network Panel

Monitor all network requests with detailed timing, headers, and response data. Filter by type, search by URL, and export for analysis.

Chrome Built-in AI

Leverage Chrome's on-device AI for private, fast analysis. Works offline with Gemini Nano for instant responses.

How It Works

Get started in minutes

Three simple steps to supercharge your debugging workflow.

01

Install the Extension

Clone the repo, run npm install && npm run build, then load the dist/ folder in Chrome's extension settings.

02

Configure Settings

Open DevTools, navigate to the DevConsole tab, and configure your AI provider, GitHub token, and other preferences.

03

Start Debugging Smarter

Click "Explain with AI" on any log, create GitHub issues instantly, explore GraphQL APIs, and enjoy intelligent autofill.

FAQ

Frequently asked questions

Everything you need to know about DevConsole.

Is DevConsole free?

Yes! DevConsole is completely open source and free to use. You can self-host it or contribute to the project on GitHub.

What AI providers are supported?

DevConsole supports OpenAI, Anthropic Claude, Google Gemini, and Chrome's built-in Gemini Nano for on-device AI processing.

Is my data secure?

Absolutely. All API keys are stored locally in your browser. With Chrome's built-in AI, your data never leaves your device.

Does it work offline?

Core features work offline. With Chrome's on-device AI enabled, even AI analysis works without an internet connection.

How do I contribute?

We welcome contributions! Check out our GitHub repo, read the ARCHITECTURE.md, and submit a PR. All skill levels welcome.

What browsers are supported?

DevConsole is a Chrome extension that works with Chrome, Chromium-based browsers, and Edge. Firefox support is planned.
